# ユーザーガイド

本ガイドでは、振動解析Webアプリケーションを効果的に使用するための詳細な手順を説明します。

## 1. はじめに

本アプリケーションは、振動データの高度な解析を行うための強力なツールです。物理的な妥当性、再現性、および拡張性に重点を置いており、製造現場や事業運用での使用に適しています。

アプリケーションを起動するには、[README.md](../README.md) のインストール手順に従った後、以下のコマンドを実行してください：

```bash
python -m streamlit run src/app.py
```

ブラウザで表示されたローカルURL（例：`http://localhost:8501`）を開きます。

## 2. データのアップロード

WAVファイル（LPCM 16/24/32bit）およびCSVファイルの両方に対応しています。**複数のファイルを同時にアップロードして一括解析を行うことも可能です。**

*   **一括削除:** 「🗑️ 全ファイルをクリア」ボタンを押すと、アップロード済みのすべてのファイルをリストから削除し、解析結果をリセットできます。

### WAVファイルの場合:
*   アップロード後、ファイル名、サンプリング周波数（Fs）、データ長が表示されます。
*   オーディオプレーヤーが表示され、録音された振動音を確認できます。

### CSVファイルの場合:
CSVファイルをアップロードすると、サイドバーに **「CSV解析設定」** が表示されます：
*   **加速度データ列を選択:** 振動データが含まれる列を選択します（複数選択可）。
*   **多軸合成を行う:** 複数列（X, Y, Zなど）を選択した場合、ベクトル合成加速度（$\sqrt{X^2 + Y^2 + Z^2}$）を計算できます。合成前に各軸のDCオフセット（重力成分など）を自動除去します。
*   **解析対象とするメインの軸を選択:** 合成を行わない場合、どの軸を特徴量抽出やMT法診断の主対象にするか選択します。他の軸は比較用にグラフの背景に表示されます。
*   **タイムスタンプ列を使用する:** タイムスタンプからサンプリング周波数を自動推論します。ISO 8601形式やUnixタイムスタンプに対応しています。
*   **サンプリング周波数 (Hz) を入力:** タイムスタンプがない場合、手動で周波数を指定します。

## 3. 解析設定とプリセット

### ⚙️ 解析プリセット
現場での運用効率を上げるため、設備ごとの解析設定を保存・呼び出しできます。
*   **プリセットを適用:** 保存済みの設定（例：「ポンプA 吐出側」）を選択して適用ボタンを押すと、フィルタや窓関数、ピーク設定が一括更新されます。
*   **現在の設定を保存:** 名前を付けて保存することで、設定を `data/analysis_presets.json` に永続化します。

### 基本設定
*   **物理量種別:** 加速度 (m/s²)、速度 (mm/s)、変位 (μm) から選択します。
*   **窓関数:** Hanning（標準）または Flat Top を選択できます。
*   **フィルタ設定:** HPF（ハイパス）と LPF（ローパス）を個別に有効化できます。設定値はFFTやスペクトログラム上の赤色点線と連動します。
*   **FFTピーク設定:** 自動アノテーションされるピークの感度を調整します。

## 4. 解析結果の解釈

### 📋 診断サマリー
複数ファイルをアップロードした場合、画面上部に全ファイルの解析結果が一覧表示されます：
*   **MD値 (異常度):** マハラノビス距離。3.0を超えると異常の兆候、10.0を超えると明らかな異常と判定されます。
*   **判定:** 🟢 正常、🟡 警告、🔴 異常 の3段階で表示されます。
*   **信頼度:** データの健全性を0-100%で示します。

### 視覚的解析（タブ切り替え）
3つのインタラクティブなチャートで詳細を確認できます：

1.  **🕒 時間領域 (波形比較):**
    *   **フィルタ適用後の信号** と **生信号（DC除去のみ）** を重ねて表示します。
    *   多軸データの場合、背景に全軸の波形が表示され、衝撃の方向などを特定しやすくなります。
    *   大規模データでもスムーズに動くよう、自動ダウンサンプリング機能が働きます。

2.  **📊 周波数領域 (FFT):**
    *   卓越周波数を自動検出し、周波数と振幅値をアノテーションします。
    *   周波数軸の **線形（Linear）/ 対数（Log）** 切り替えが可能です。
    *   フィルタの遮断周波数が赤色の垂直線で表示されます。

3.  **🌈 時間-周波数領域 (スペクトログラム):**
    *   周波数成分の経時変化を可視化します。
    *   窓長（Window Size）を調整して、時間・周波数の解像度をバランスさせることができます。
    *   各軸個別のスペクトログラムを俯瞰し、特定の軸にのみ発生するイベントを確認できます。

### データ品質と信頼度
*   **クリッピング率:** センサの測定範囲を超えて波形が潰れた割合。
*   **S/N 比:** ノイズに対する信号の強さ。
*   **🤔 診断信頼度:** クリッピング、S/N比、データ長に基づく総合スコア。80%以上が信頼できる目安です。

## 5. MT法（マハラノビス・タグチメソッド）

異常診断を行うには、まず正常データによる「単位空間」の構築が必要です。
1.  サイドバーの **「MT法設定」** を展開します。
2.  正常時のファイルを10個以上（30個以上推奨）アップロードします。
3.  **「単位空間を構築/更新」** をクリックします。これで、以降の解析データに対してMD値が算出されるようになります。

## 6. 監査ログ (Audit Log)

解析の再現性を確保するため、すべてのパラメータと結果が画面下部にJSON形式で表示されます。このログはファイルとしてダウンロードでき、後で全く同じ条件で再計算することが可能です。
